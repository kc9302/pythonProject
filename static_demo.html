                        movie Media Profile (10종)
                    

                    

                        

                            
verb_count Basic

                            
유저의 미디어 특정 verb(played, paused 등)의 발생 빈도수를 구합니다.

                            
Input: name, verbReturn: int

                        

                        

                            
watched_media_list Basic

                            
유저가 시청한 미디어 명칭 목록과 영상/오디오 등 미디어 유형을 구합니다.

                            
Input: datasetReturn: dict

                        

                        

                            
watched_media_count Basic

                            
유저의 영상 및 오디오 미디어 유형별 시청 횟수(Played 빈도)를 구합니다.

                            
Input: name, verbReturn: dict

                        

                        

                            
initialized_info Basic

                            
미디어 초기화 시 확장 데이터(길이, 세션 ID, 프레임레이트 등 15종)를 구합니다.

                            
Input: datasetReturn: dict

                        

                        

                            
now_play_time Basic

                            
미디어 재생 시작 시점(extensions.time)을 미디어별로 파악합니다.

                            
Input: datasetReturn: dict

                        

                        

                            
play_pause_time_interval Basic

                            
미디어별 재생 및 일시정지 시간 구간 정보와 세그먼트별 진도를 계산합니다.

                            
Input: datasetReturn: dict

                        

                        

                            
seeked_time_interval Basic

                            
유저의 미디어 타임라인 탐색(Seeked) 시작 위치와 도달 위치 구간을 추적합니다.

                            
Input: datasetReturn: dict

                        

                        

                            
completed_time_interval Basic

                            
유저의 미디어 시청 완료율 및 완료 처리 이후 시청한 추가 진도 구간을 계산합니다.

                            
Input: datasetReturn: dict

                        

                        

                            
terminated_time_interval Basic

                            
미디어 재생 종료(Terminated) 시점의 위치 정보와 최종 종료 상태를 파악합니다.

                            
Input: datasetReturn: dict

                        

                        

                            
interacted_time_interval Basic

                            
미디어 내 인터랙티브 요소(자막 등) 제어 구간 정보와 제어 항목을 추출합니다.

                            
Input: datasetReturn: dict

                        

                        

                            
media_heatmap Applied

                            
특정 미디어 콘텐츠(activity_id) 구간별 pause 및 seek 발생 빈도를 5초 간격으로 분석하여 인지 병목 구간을 진단합니다.

                            
Input: activity_idReturn: list

                        assignment_turned_in Assessment Profile (5종)
                    

                    

                        

                            
verb_count Basic

Input: name, verbReturn: int

                        

                        

                            
solved_question_count Basic

                            
유저가 완수/풀이한 평가 문항 목록과 문항별 최종 풀이 완료 횟수를 구합니다.

                            
Input: datasetReturn: dict

                        

                        

                            
attempt_question_count Basic

                            
유저가 풀이 시도한 평가 문항 목록과 문항별 단순 총 시도(Attempt) 횟수를 구합니다.

                            
Input: datasetReturn: dict

                        

                        

                            
extensions Basic

                            
평가 문항의 컨텍스트 확장 메타정보(학년, 과목, 학기, 태그, 평가 유형)를 추출합니다.

                            
Input: datasetReturn: dict

                        

                        

                            
interaction Basic

                            
유저가 풀이한 평가 문항의 객관식/주관식 등 인터랙션 유형 정보를 구합니다.

                            
Input: datasetReturn: dict

                        

                        

                            
assessment_efficiency Applied

                            
문항별 획득 점수와 소요 시간을 초 단위로 계산하여 효율 지수를 도출하고 행동 패턴(찍기 의심 등)을 진단합니다.

                            
Input: user_idReturn: list

                        percent 학습 정답률 통계 (4종)
                    

                    

                        

                            
assessment_history Applied

                            
전체 기간의 학년별, 과목별 평균 정답률 및 전체 평균 정답률을 계산합니다.

                            
Input: datasetReturn: dict

                        

                        

                            
assessment_grade_history Applied

                            
특정 학년에 한하여 과목별 평균 정답률과 전체 평균 정답률을 파악합니다.

                            
Input: dataset, gradeReturn: dict

                        

                        

                            
subject_correct_rate Applied

                            
특정 학년과 과목에서 세부 단원(태그)별 평균 정답률과 과목 평균 정답률을 구합니다.

                            
Input: dataset, grade, subjectReturn: dict

                        

                        

                            
assessment_tag_correct_rate Applied

                            
특정 단원(태그)에 소속된 개별 평가 문항들의 개별 상세 정답 횟수 및 정답률을 추출합니다.

                            
Input: dataset, grade, subject, tagReturn: dict

                        

                    


                    

                        menu_book 오답 분석 및 상세 목록 (3종)
                    

                    

                        

                            
wrong_answers Applied

                            
유저의 오답 목록을 문항 ID, 문항 명칭, 단원명, 그리고 동일 문항 누적 시도 횟수와 함께 제공합니다.

                            
Input: datasetReturn: dict

                        

                        

                            
wrong_answers_assessment_type Applied

                            
특정 평가 유형(예: 차시평가, 단원평가)에 필터링된 오답 목록 및 총 시도 횟수를 구합니다.

                            
Input: dataset, assessment_typeReturn: dict

                        

                        

                            
wrong_answers_tag Applied

                            
특정 개념(태그)에 필터링된 오답 목록과 총 시도 횟수를 추출합니다.

                            
Input: dataset, tagReturn: dict

                                    🟡 테스트 계정 데이터 (Tester)
